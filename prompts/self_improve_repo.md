# self_improve_repo

Analyze `reels-automation-agent` as a self-improving development repository.

## Objective

Strengthen the repository's own operating system:

- AI prompt quality
- issue generation quality
- patch bundle safety
- PR automation clarity
- workflow observability

## Focus Areas

- whether agent outputs are structured enough for automation
- whether issue and PR policy are strict enough
- whether branch / commit / PR creation remains human-safe
- whether scripts and workflows are simple, debuggable, and local-friendly

## Constraints

- preserve human approval before merge
- keep automatic changes small
- avoid broad automation that can rewrite the product dangerously

## Expected Outcome

- better automation prompts
- safer patch filtering
- clearer workflow logs and artifacts
- better docs for future maintainers
